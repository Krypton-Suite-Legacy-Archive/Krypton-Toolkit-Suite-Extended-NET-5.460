KryptonVirtualListBox
A simple implementation of a Virtual ListBox. A virtual ListBox contains no data, instead it just allocates space for a specified number of rows. Whenever a row needs to be shown, the [!:OnDrawItem] method is fired which in turn fires the DrawItem(Object, DrawItemEventArgs) event.
