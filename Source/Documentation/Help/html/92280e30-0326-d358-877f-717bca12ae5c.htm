Style Fields

The Style type exposes the following members.

Fields
  NameDescription
BraceBad
Bad brace style index. This style is used on an unmatched brace character when set with the BraceBadLight(Int32) method.
BraceLight
Brace highlighting style index. This style is used on a brace character when set with the BraceHighlight(Int32, Int32) method or the indentation guide when used with the HighlightGuide property.
CallTip
Call tip style index. Only font name, size, foreground color, background color, and character set attributes can be used when displaying a call tip.
Default
Default style index. This style is used to define properties that all styles receive when calling StyleClearAll.
FoldDisplayText
Fold text tag style index. This is the style used for drawing text tags attached to folded text when FoldDisplayTextSetStyle(FoldDisplayText) and ToggleFoldShowText(String) are used.
IndentGuide
Indent guide style index. This style is used to specify the foreground and background colors of IndentationGuides.
LineNumber
Line number style index. This style is used for text in line number margins. The background color of this style also sets the background color for all margins that do not have any folding mask set.
