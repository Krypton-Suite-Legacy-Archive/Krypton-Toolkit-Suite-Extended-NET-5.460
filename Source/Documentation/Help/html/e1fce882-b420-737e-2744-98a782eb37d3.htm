TryParseExact(String, String, IFormatProvider, TimeSpan2)
Converts the string representation of a time interval to its TimeSpan2 equivalent by using the specified format and culture-specific format information, and returns a value that indicates whether the conversion succeeded. The format of the string representation must match the specified format exactly.
