result
Type: SystemTimeSpan2
When this method returns, contains the TimeSpan2 value equivalent to the time span contained in s, if the conversion succeeded, or TimeSpan.Zero if the conversion failed. The conversion fails if the s parameter is null, is an empty string (""), or does not contain a valid string representation of a time span. This parameter is passed uninitialized.

Return Value

Type: Boolean
true if the s parameter was converted successfully; otherwise, false.
