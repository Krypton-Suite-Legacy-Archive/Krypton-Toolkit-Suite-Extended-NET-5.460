Parameters

level
Type: SystemInt32
The level of the line to search for. A value of -1 will use the current line FoldLevel.

Return Value

Type: Int32
The zero-based index of the next line that has a FoldLevel less than or equal to level. If the current line is a fold point and level is -1 the index returned is the last line that would be made visible or hidden by toggling the fold state.
